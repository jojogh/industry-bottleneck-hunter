7. **欧洲经济疲软**：Ovako销量大幅下滑101千吨

## 三、轴承钢业务——谐波减速器产业链核心

### 3.1 山阳轴承钢业务地位

| 指标 | 数据 |
|------|------|
| **日本轴承钢市占率** | **约37%（日本第一）** |
| **轴承钢年产量** | **超过30万吨** |
| **日本轴承钢总产量** | 约95万吨/年 |
| **历史积淀** | 1935年开始生产轴承钢，近90年技术积累 |
| **产品主力牌号** | SUJ2（对应中国GCr15、德国100Cr6、美国52100） |

### 3.2 超纯净钢技术——核心竞争力

#### 3.2.1 技术发展历程

| 年代 | 技术阶段 | 代表产品 | 氧含量水平 |
|------|----------|----------|------------|
| **1980年代** | 高纯净度钢 | SP钢 | ~50 ppm → 逐步降低 |
| **1990年代** | 超高纯净度钢 | EP钢 | **≤5 ppm** |
| **2018年** | 极超纯净度钢 | Premium J2钢 | **2~3 ppm** |

#### 3.2.2 SNRP工艺（Sanyo New Refining Process）

**流程**：90t UHP电弧炉 → 偏心炉底出钢(EBT) → LF精炼 → RH真空脱气 → 完全垂直连铸

| 关键控制指标 | 数值 |
|-------------|------|
| **氧含量** | **5 ppm（5×10⁻⁶）** |
| **夹杂物尺寸** | **≤11 μm** |
| B类夹杂物级别 | 0.06级 |
| D类夹杂物级别 | 0.06级 |

#### 3.2.3 SURP工艺（Sanyo Ultra Refining Process，2018年开发）

- 极超纯净度钢生产工艺
- 控制夹杂物的应力、变性
- 控制大型夹杂物出现频率
- 控制夹杂物的种类、数量、比例
- **氧含量最低可达2~3 ppm**

#### 3.2.4 与传统工艺对比

| 对比项 | SNRP超纯净钢（连铸） | 传统工艺（模铸） | 提升幅度 |
|--------|---------------------|------------------|----------|
| **氧含量** | 5 ppm | ~7.6 ppm | 降低34% |
| **B类夹杂物** | 0.06级 | 0.4级 | **降低至1/7** |
| **D类夹杂物** | 0.06级 | 0.8级 | **降低至1/13** |
| **接触疲劳寿命** | 延长5倍 | 基准 | **500%提升** |
| **与VAR法对比** | 氧含量和寿命相同 | — | 成本更低 |

### 3.3 谐波减速器轴承钢材料要求

#### 3.3.1 谐波减速器核心材料应用

| 部件 | 典型材料 | 材料类别 |
|------|----------|----------|
| **波发生器内外圈** | ZGCr15 | 高碳铬轴承钢 |
| **滚动体** | GCr15 | 高碳铬轴承钢 |
| 柔轮 | 40CrNiMoA | 合金结构钢 |
| 刚轮 | 40CrMo | 合金结构钢 |
| 凸轮 | 45# | 优质碳素结构钢 |

#### 3.3.2 谐波减速器轴承钢"变态级"要求

| 性能维度 | 具体要求 |
|----------|----------|
| **纯净度** | 非金属夹杂物控制在微米级 |
| **氧含量** | **≤8 ppm（高端要求≤5 ppm）** |
| **夹杂物** | **≤5 ppm（TiN/Al₂O₃严控）** |
| **均匀度** | 整个截面硬度波动不能超过1 HRC |
| **表面质量** | GCr15轴承钢表面粗糙度Ra 0.4μm |
| **疲劳寿命** | 数万小时（交变应力环境） |

#### 3.3.3 国内外差距

| 对比维度 | 国内现状 | 国际水平（日本/SKF） |
|----------|----------|---------------------|
| 材料提纯技术 | 存在差距，杂质较多 | 超纯净钢冶炼技术成熟 |
| 主要失效形式 | 齿轮磨损（提前失效） | 柔性轴承破坏（设计寿命终点） |
| 供应链地位 | 主要依赖进口 | 垄断高端市场 |
| 典型氧含量 | 8-12 ppm | **≤5 ppm** |

## 五、技术护城河

### 5.1 超低含量非金属夹杂物高纯净度钢技术

山阳特殊制钢的核心技术护城河在于其**系统性夹杂物控制技术**，而非单一指标：

```
LF-RH精炼工艺 + 全程保护浇铸 + 立式连铸
    ↓
氧含量 ≤ 5 ppm（最低2~3 ppm）
    ↓
夹杂物尺寸 ≤ 11 μm
    ↓
B类夹杂物 0.06级（传统工艺的1/7）
    ↓
D类夹杂物 0.06级（传统工艺的1/13）
    ↓
接触疲劳寿命延长5倍
```

### 5.2 氧、硫、磷含量控制

| 元素 | 控制水平 | 工艺手段 |
|------|----------|----------|
| **氧(O)** | **≤5 ppm（典型）/ 2~3 ppm（最优）** | LF精炼+RH真空脱气+软吹氩 |
| **硫(S)** | 极低水平 | 高碱度渣（CaO/SiO₂=4.5~7.2） |
| **磷(P)** | 极低水平 | 精炼渣系优化 |
| **钛(Ti)** | ≤5 ppm | 原料控制+精炼工艺 |
| **氮(N)** | ≤30 ppm | 真空脱气+保护浇铸 |

### 5.3 合金元素偏差控制

| 控制维度 | 技术水平 |
|----------|----------|
| 成分均匀性 | 整炉成分偏差极小 |
| 硬度均匀性 | 整个截面硬度波动≤1 HRC |
| 组织均匀性 | 贝氏体/马氏体组织精确控制 |
| 表面质量 | Ra ≤ 0.4μm |

### 5.4 精炼渣系核心技术

| 参数 | 最优范围 | 效果 |
|------|----------|------|
| **碱度(CaO/SiO₂)** | 4.5~7.2（最优）/ 最高10.3 | 高碱度显著减少大型夹杂物 |
| **氧化性(FeO+MnO)** | ≤0.5% | 降低二次氧化 |
| **LF精炼时间** | >45分钟 | 充分产生MgO系夹杂→转变为CaO系 |
| **RH真空时间** | 40~60分钟 | 夹杂物总量降至最低 |
